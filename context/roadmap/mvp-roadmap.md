# MVP_ROADMAP.md

# Jknowledge MVP

## Goal

Build a simple demo for:
- TCAS score analysis
- Admission prediction
- MBTI faculty recommendation

Focus on:
- fast development
- good UI
- mobile-first
- shareable results

---

# Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Recharts
- Vercel

---

# Phase 1 — Setup

Tasks:
- Setup Next.js
- Install Tailwind
- Install shadcn/ui
- Setup TypeScript
- Create folder structure
- Create landing page

---

# Phase 2 — Mock Data

Create mock TCAS score data first.

Example:
- university
- faculty
- min score
- avg score
- yearly scores

Do NOT use database yet.

---

# Phase 3 — Analyze Feature

Create:
- /analyze page

Features:
- Input TCAS scores
- Select university/faculty
- Calculate admission chance
- Show result card

Simple logic:
- High Chance
- Competitive
- Low Chance

---

# Phase 4 — Charts

Add:
- 5-year score trend chart
- Progress bars
- Status badges

Use Recharts.

---

# Phase 5 — MBTI Demo

Create:
- simple MBTI quiz
- faculty recommendations

Focus on:
- UX
- fun
- shareability

---

# Phase 6 — Deploy

Deploy MVP using Vercel.

Goals:
- mobile responsive
- fast loading
- easy sharing

---

# Important Rules

Never:
- guarantee admission
- fake official statistics

Always:
- use estimate-based wording
- prioritize mobile UX
- keep UI simple
