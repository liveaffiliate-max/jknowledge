  - อัพเดท `normalizeMajor()`: handle "SUBJECT วิชาเอกSPEC" pattern (mid-string)

    - "ใช้คะแนน..." → "" (filtered)
    - "คณะX เกษตรศาสตร์ วิชาเอกพืชสวน" → "" (contains วิชาเอก)
    - "โครงการปกติ" → "โครงการปกติ" ✓ (kept)
  - Apply `normalizeProgram()` ใน map ของ 3 functions: `getFacultiesByUniSlug`, `getFacultiesByUniversityId`, `getFacultyBySlug`, `getFacultyWithScores`
